# papers — PDF corpus

Place journal PDFs here (including subfolders). Each file’s **`source_path` in the manifest** must be the full repo-relative path (for example `papers/Others/foo.pdf`, not just `foo.pdf`). See [`AGENTS.md`](../AGENTS.md).

**Git:** Large PDFs may be excluded from version control; see `.gitignore` and the “Git, privacy, and public export” section in `AGENTS.md`. If PDFs are not tracked, each clone must restore files so `source_path` in the manifest resolves on disk.
