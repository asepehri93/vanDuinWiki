# Phase 8 — Public export (outline)

**Phase 8 is not** theme expansion, debates, or retrieval tuning. Those stay under **Phase 4** / **Phase 5** per [`MASTER_PLAN.md`](MASTER_PLAN.md).

Phase 8 answers: *what subset of this repo may ship to a public site or third parties* without violating rights, privacy, or lab policy.

## Checklist (operator fills in)

- [ ] **Scope:** Which paths are exportable (`wiki/` subset only? exclude `normalized/`?).
- [ ] **PDFs:** Default is **do not** publish full PDFs; link to DOIs or omit.
- [ ] **Redaction:** Strip internal paths, student names, or grant numbers if required.
- [ ] **License:** Confirm publisher / author permissions for any excerpt beyond short citation.
- [ ] **Machine layer:** If publishing derived JSON or chunks, document provenance and refresh policy.
